# Цукиба
Имиджборд для анонимных сетей

Для запуска требуется: PHP 8+, PHP GD, FFmpeg и ExifTool\
Найдите фразу 'secretkey' в файле index.php и замените на любую подходящую\
Перейдите по адресу /manage?key=ваш_ключ и настройте сайт на свой вкус

**Данный имиджборд подходит только для анонимных сетей**
